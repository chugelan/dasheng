## 五、已知问题

### 1. better-sqlite3 编译失败

**问题：** Node.js v25.8.0 太新，导致 native 模块编译失败

**解决方案：** 
- ✅ 已改用 JSON 文件存储（简化版数据库）
- 备选：降级到 Node.js v20 LTS

### 2. yargs 模块警告

**问题：** 依赖库使用 require 语法与 ES modules 不兼容

**影响：** 无实际影响，可忽略
